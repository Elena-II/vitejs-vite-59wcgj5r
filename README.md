# Тренировочные тесты для 2 класса

Приложение для прохождения школьных тренировочных тестов. Полностью соответствует техническому заданию.

## Технологии
- React + TypeScript
- Vite
- React Router
- Dexie (IndexedDB)
- Progressive Web App (PWA)

## Основные возможности
- Полностью работает оффлайн после первой загрузки
- Автосохранение прогресса
- Продолжение незавершённого теста
- Статистика по предметам
- Разбор ошибок с подсказками
- Адаптивный дизайн для детей

## Структура предметов
Все данные хранятся в папке `public/subjects/` в формате JSON.

## Запуск проекта

```bash
npm install
npm run dev
